# Water Quality Machine Learning Pipeline

## Overview
This project builds a complete machine learning pipeline to analyze and predict water quality using multiple models. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, and model evaluation.

## Features
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection and preprocessing
- Model training and evaluation
- Comparison of multiple ML algorithms
- Export of results to CSV

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Workflow
1. Load dataset
2. Perform EDA
3. Handle missing values
4. Feature scaling and transformation
5. Train multiple models
6. Evaluate model performance
7. Compare results
8. Save predictions/output

## Requirements
- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Install dependencies:
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## Usage
Run the Jupyter Notebook:

```bash
jupyter notebook Water_Quality_ML_Pipeline.ipynb
```

## Output
- Model performance metrics
- Comparison of algorithms
- CSV file with predictions/results
